MEDIA FILES FOR CUCUTA BILINGUAL BROCHURE
=========================================

Place the following files in this folder:

1. video1.mp4
   - Project overview video
   - Used on: Cover page (Page 1)
   - Recommended: 16:9 aspect ratio, max 50MB

2. image1.jpg
   - Q-UGV robot photo (quadrupedal robot)
   - Used on: Page 2 hero section
   - Recommended: 1200x512px or similar landscape

3. image2.jpg
   - Sewer cleaning robot photo
   - Used on: Page 3 hero section
   - Recommended: 1200x480px or similar landscape

4. image3.jpg
   - Team/equipment photo
   - Used on: Page 4 (alliance section)
   - Recommended: 800x200px or similar wide format

5. cucuta-proposal.pdf (OPTIONAL)
   - Pre-generated PDF version of the brochure
   - If this file exists, the download button will serve it
   - If not, the download button will trigger browser print

NOTES:
- All images should be optimized for web (compressed JPEGs)
- Video should be MP4 format with H.264 codec for best compatibility
- Keep total media size under 100MB for fast loading
